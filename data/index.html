Quick Actions

Steppers stay energized (holding torque) by default. Disable to release them so the rig can be moved by hand; any move re-enables them. Re-Home Linear runs the limit-switch homing to find the linear 0; Re-Home Rotary sets the current pan angle as 0°.

Linear Axis LINEAR

mm the carriage travels per full motor turn. Jog speed sets how fast the Left/Right buttons move. Flip direction if LEFT/RIGHT or homing run the wrong way.

Rotary Axis ROTARY

Steps for one full platform turn. Direct drive on a 1/16 NEMA 17 = 3200; raise it for a geared head.

Other Settings

Smooth ramping eases both axes in and out of motion (jog and auto). Turn it off for snappy, constant-speed moves with no acceleration.
